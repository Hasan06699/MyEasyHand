# MyEasyHand Employee App — myeasyhand-employee-app

Employee mobile application for the MyEasyHand Service Booking Platform.

| Property | Value |
|---|---|
| **Platforms** | iOS, Android |
| **Stack** | React Native, Expo, TypeScript, Redux Toolkit, React Query, OneSignal |

## Quick Start

```bash
cp .env.example .env
npm install
npx expo start
```

## Features (Planned)

- View assigned bookings
- Update booking status
- Manage availability & schedule
- Push notifications (assignment alerts)
- Profile management

## Branch Strategy

```
development → staging → main
```
