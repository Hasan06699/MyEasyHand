# MyEasyHand Admin — myeasyhand-admin

Super Admin and Business Owner management panel for the MyEasyHand platform.

| Property | Value |
|---|---|
| **Port** | 8081 |
| **URL** | https://admin.myeasyhand.in |
| **Template** | MaterialPro Next.js Free |
| **Stack** | Next.js, TypeScript, MUI, React Query, Zustand |

## Quick Start

```bash
cp .env.example .env
npm install
npm run dev
```

Open http://localhost:8081
